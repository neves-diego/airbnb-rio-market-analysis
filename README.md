# airbnb-rio-market-analysis
Análise de dados do Airbnb no Rio de Janeiro com foco na geração de insights para o mercado imobiliário. O projeto explora padrões de precificação, competitividade entre bairros e indicadores de ocupação, demonstrando como dados públicos podem apoiar decisões de investimento e planejamento estratégico.

📊 Airbnb Rio – Análise de Mercado Imobiliário

📌 Sobre o Projeto

Este projeto realiza uma análise exploratória dos dados públicos do Airbnb no Rio de Janeiro, com foco na geração de insights aplicáveis ao mercado imobiliário.

A proposta é transformar dados brutos em informações estratégicas que possam apoiar decisões de investimento, precificação e posicionamento comercial.

🎯 Objetivo

Desenvolver uma análise clara, replicável e orientada a negócio para:

Identificar bairros com maior potencial de rentabilidade

Avaliar nível de concorrência por região

Comparar desempenho entre tipos de imóveis

Estimar indicadores indiretos de ocupação

Criar um índice próprio de atratividade comercial

🛠 Ferramentas Utilizadas

Python

Pandas

Matplotlib

Jupyter Notebook

🧹 Etapas do Projeto

Importação dos dados públicos do portal Inside Airbnb

Seleção de variáveis estratégicas

Limpeza e padronização da coluna de preços

Tratamento de valores ausentes

Análise exploratória com foco comercial

Criação de métrica de atratividade por bairro

📊 Principais Análises
🔹 Preço médio por bairro

Identificação de regiões com maior ticket médio.

🔹 Volume de anúncios

Avaliação da concentração e competitividade do mercado.

🔹 Rentabilidade por tipo de imóvel

Comparação entre imóvel inteiro, quarto privado e outros formatos.

🔹 Indicador indireto de ocupação

Análise baseada na disponibilidade anual informada na plataforma.

🔹 Índice de Atratividade Comercial

Métrica desenvolvida a partir da relação entre preço médio e disponibilidade média, permitindo gerar um ranking estratégico de bairros.

💼 Aplicabilidade

O modelo pode ser adaptado para qualquer cidade onde os dados do Airbnb estejam disponíveis, permitindo que imobiliárias e investidores utilizem dados públicos como ferramenta estratégica de tomada de decisão.
